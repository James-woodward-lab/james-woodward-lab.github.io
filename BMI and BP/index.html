BMI & Blood Pressure Centile Calculator

For children aged 2–18 years. Growth calculations use the RCPCH Digital Growth API when available. % median BMI is read directly from the RCPCH BMI API response when available. BP centiles are estimated from embedded Jackson 2007-derived testing tables for ages 4–18.
